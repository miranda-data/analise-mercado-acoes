# Análise de Dados - Mercado de Ações

Projeto de análise de dados realizado em planilha, com base em um conjunto simulado de operações no mercado de ações.

## Objetivo

Explorar dados financeiros para extrair métricas estatísticas, identificar tendências temporais e analisar o comportamento de clientes em operações de compra e venda.

## Etapas do Projeto

### 1. Tratamento de Dados

* Remoção de duplicidades
* Tratamento de valores nulos
* Padronização dos dados

### 2. Análise Estatística (Preço)

* Soma
* Mínimo e máximo
* Média, mediana e moda
* Variância e desvio padrão

### 3. Tendência Temporal

* Análise mensal das operações
* Projeção de valores futuros
* Estimativa para o mês de dezembro com base nos dados históricos

### 4. Resumo Geral

* Total de operações de compra e venda
* Consolidação dos dados da base principal

### 5. Análise por Cliente

* Quantidade de operações por cliente
* Volume financeiro movimentado por cliente

### 6. Lucro e Prejuízo por Cliente

* Análise filtrada por cliente específico
* Avaliação de desempenho em ativos selecionados

### 7. Visualização de Dados

* Gráficos individuais por análise
* Consolidação dos principais gráficos em uma aba dedicada

## Ferramentas Utilizadas

* Microsoft Excel / Google Sheets

## Observação

Projeto desenvolvido como parte dos meus estudos em análise de dados, com foco em prática de manipulação, análise e interpretação de dados.
